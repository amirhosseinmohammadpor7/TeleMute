# [TeleMute](https://github.com/amirhosseinmohammadpor7/TeleMute)
*TG-CLI based broadcasting bot!*
# Install
```
cd $HOME
git clone https://github.com/amirhosseinmohammadpor7/TeleMute.git
cd TeleMute
chmod +x install.sh
chmod +x launch.sh
chmod +x tg
./install.sh
./launch.sh
# Enter a phone number & confirmation code.
```
Create a bot!
